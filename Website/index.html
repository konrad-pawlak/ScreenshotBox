How to use it?

Jak użyć?



1. Run it

1. Uruchom exe

Download, unzip it and place in any folder you like. This folder will be used to store your printscreens.

Pobierz exe i umieść w wybranym folderze. Do tego folderu będą od teraz zapisywane wszustkie zrzuty ekranu jakie zrobisz.

2. Press PrtScr key

2. Naciśnij PrtScr na klawiaturze

You can also press Alt+PrtScr to make a screenshot of a window.

Możesz użyć również kombinacji Alt+PrtScr aby, zamiast całego ekranu, zrobić zrzut aktywnego okna.

3. Done!

3. Gotowe!

The printscreen is saved in the same folder where you placed ScreenshotBox.exe

Zrzut ekranu jest w Twoim folderze. Możesz do niego wrócić kiedy zechcesz.

Add it to autostart!

Dodaj do autostartu!

The power of this app is that it works in the background!

Co jest najlepsze ScreenshotBox'ie? To, że możesz o nim zapomnieć.

You don't need to remember to set it up. If you notice something worth saving, just press PrintScreen key. You can come back to the screenshot at any time you want.

Nie musisz go włączać albo specjalnie ustawiać przed użyciem. Gdy widzisz coś wartego zapisania, klikasz klawisz PrtScn. Możesz wrocić do zapisanego zrzutu kiedy tylko chcesz.

How?

Jak?

Simply create a shortcut to the ScreenshotBox.exe and place it in following folder:

Zrób skrót do ScreenshotBox.exe i umieść go w folderze:

C:\Users\#your user name##twoja nazwa użytkownika#\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup
